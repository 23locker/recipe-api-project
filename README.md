# Recipe API

## О проекте

Бэкенд приложение для работы с рецептами на FastAPI. Можешь просматривать рецепты, фильтровать по калориям и категориям, заменять ингредиенты если их нет, парсить цены из магазина Пятерочка.

## Что внутри

- **FastAPI** - вебфреймворк
- **Tortoise ORM** - работа с PostgreSQL
- **Motor** - асинхронный драйвер для MongoDB
- **JWT** - авторизация
- **Redis** - кэширование рецептов
- **Docker** - контейнеризация

## Структура БД

**PostgreSQL:**

- `users` - пользователи (user, admin)
- `categories` - категории рецептов
- `ingredients` - справочник ингредиентов
- `substitutes` - замены ингредиентов

**MongoDB:**

- `recipes` - рецепты с вложенными ингредиентами и шагами
- `store_products` - товары из магазинов
